开始之前

相信来到这里的各位都知道hugo（中文版)是什么了吧。 什么，你不知道？那么请阅读以下内容：

Hugo是由Go语言实现的静态网站生成器。
简单、易用、高效、易扩展、快速部署。
对比Hexo，Hugo仅需一个二进制文件（hugo.exe）即可实现网站生成的生成。
而且就如它官网介绍：
The world’s fastest framework for building websites。
且Hugo是目前最快的网站构建框架。
凭借其惊人的速度和灵活性，解决了环境依赖、性能较差的问题。
还有使用简单、部署方便等诸多优点。
通过 LiveReload 实时刷新，极大的优化文章的写作体验。
Hugo使建站再次变得有趣。

看到了吗，这就是hugo，好了，那就让我们开始建站吧！

下载hugo

hugo在windows下一共有2种下载方式：

1、github

选择合适自己电脑的版本下载即可

2、代码实现（Chocolatey）

按下Windows+R键打开命令行，在里面输入

1
2
3
4
5

powershell -NoProfile -ExecutionPolicy unrestricted -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))"

SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin

choco install hugo -confirm

即可安装hugo

*配置hugo路径

配置路径是至关重要的一点，不然要使用hugo命令就只能使用完整命令

配置路径具体方法如下：

  • 1、右键此电脑，选中属性
  • 2、点击系统高级设置
  • 3、点击环境变量
  • 4、在系统变量中找到path，再点击编辑
  • 5、点击新建，填入hugo解压的文件目录

做完这一切后，打开命令行，输入hugo version来检查你是否成功安装hugo

新建博客

首先按下Windows+R键打开命令行，cd到你想建站的目录下
使用命令hugo new site [mysite]

*注意，这里的[mysite]换成你所建的站的名字

然后在你的指定文件夹下就新建了了一个hugo博客 目录结构如下：

├── archetypes
│   └── default.md
├── content             # 博客文章所在目录
├── data                
├── layouts             # 网站布局
├── static              # 一些静态内容
├── themes              # 博客主题
└── config.toml         # 博客站点的配置文件

下载主题

hugo有几百种不同的主题，你可以自行选择。

选择完后就进行安装

安装方式很简单，几行代码就可以搞定:

*注意，以下所有的代码都在命令行中实现

1、 cd theme

2、 git clone [website] *[website]为你选择的主题在github上的链接(https://github.com/Name/example.git)

3、 按照你选择的主题教程安装剩下部分

新建文章

既然要做博客，自然要写文章

hugo支持.md文件转.html文件，所以很方便，但你需要有一定markdown基础

步骤：

1、 cd ./

2、 mkdir content\posts

3、 hugo new posts\[name].md *[name]为文章名

4、 notepad content\posts\[name].md

5、 开始书写文章！

*需要注意的是，文章上方会有---或+++等，它包括了提交文档的主要数据，例如名称、时间等 在 Hugo 中，这叫做前缀 在前缀之后，才是正文.下面编辑第一个提交文件内容：

1
2
3
4
5
6

---
title: "Test"
date: 2020-12-04T23:04:22+08:00
draft: true
---
Hugo Test!

6、 写完后保存，然后hugo server -D将其配置到浏览器上，然后在浏览器里输入http://localhost:1313以打开你的博客
